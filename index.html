Ajuda - Sistema de Relatório de Turno

                  
              

              

                  

                      
Como usar:

                      
    
                          
  1. Preencha as Informações do Turno (Data, Horário, Letra, Supervisor).
    2. 
                          
  2. Clique em "Próximo".
    3. 
                          
  3. Adicione as equipes de Alta Pressão e/ou Vácuo/Hiper Vácuo clicando nos botões correspondentes.
    4. 
                          
  4. Preencha os detalhes de cada equipe no formulário que aparecerá (incluindo Tipo e Status da Atividade).
    5. 
                          
  5. Se o Status não for "Concluído", preencha a Justificativa/Pendência.
    6. 
                          
  6. Se houver troca de equipamento, marque "Sim" e preencha os detalhes obrigatórios.
    7. 
                          
  7. Após adicionar todas as equipes, clique em "Próximo: Revisão".
    8. 
                          
  8. Confira todos os dados no resumo.
    9. 
                          
  9. Clique em "Salvar Relatório".
    10. 
                          
  10. Após salvar, você poderá visualizar o relatório, formatá-lo para WhatsApp ou gerar um PDF (se salvo no servidor).
    11. 
                          
  11. Use o botão "Pesquisar" no topo para encontrar relatórios antigos.
    12. 
                          
  12. Use o botão "Dashboard" para ver estatísticas (requer login Google, se configurado).
    13. 
                      

                      
Solução de Problemas:

                      
    
                          
  • Se as opções nos menus não carregarem, verifique sua conexão com a internet. O sistema usará opções padrão se a conexão falhar.
    • 
                          
  • Se o salvamento falhar, o sistema tentará salvar localmente no seu navegador. Relatórios locais podem ser pesquisados usando a opção "Relatórios Locais".
    • 
                          
  • Para erros persistentes, contate o desenvolvedor (Warlison Abreu).
    • 
                      

                      
Dicas:

                      
    
                          
  • Use o botão "Modo Escuro" para alternar o tema visual.
    • 
                          
  • Relatórios salvos localmente só ficam disponíveis neste navegador.
    •
